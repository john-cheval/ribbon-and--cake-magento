Baking Classes Dev CMS Editing Guide

Frontend styling is controlled in GraphCommerce CSS. In Magento, edit the labelled HTML Code block directly below each admin note. Keep class names and data markers unchanged unless frontend CSS is updated too.

  • Each section has an admin label such as “01 Hero Banner” followed by one HTML Code block.
  • Links are marked with data-cms-link. Change the href on the same tag.
  • Images are normal HTML image tags. Replace only the image path and alt text values.
  • The booking form fields are managed under Content > Elements > Manage Custom Forms > Baking Classes Dev Booking. Keep data-baking-custom-form="baking-classes-dev-booking" unchanged so GraphCommerce can mount it.
  • This page is independent from the existing live /baking-classes page.
